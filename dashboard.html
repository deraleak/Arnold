// TIPS
Wake word required — say "Hey Arnold" to activate the 8-second command window. The mic bar will turn cyan and count down. Nothing is processed until the wake word is heard.

Always say the full exercise name — voice logging requires the exercise name to be spoken. Shorthand like "315 x 5" alone will be rejected.

Confidence filtering — commands containing numbers require 90%+ speech recognition confidence. Low-clarity audio is dropped and shown as LOW CONF.

Cancel — say "cancel" or "never mind" to exit the command window early and return to wake word mode.

10-second cooldown — after any numeric voice command is accepted, the mic ignores further numeric input for 10 seconds to prevent duplicate logs.
